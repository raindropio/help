Raindrop.io is free to use with no limits on bookmarks, collections, or devices. A Pro subscription unlocks additional storage, features, and tools.

## Comparing plans

| | Free | Pro |
| --- | --- | --- |
| Bookmarks | Unlimited | Unlimited |
| Collections | Unlimited | Unlimited |
| Tags | Unlimited | Unlimited |
| Collaborators per collection | Unlimited | Unlimited |
| Sync devices | Unlimited | Unlimited |
| Highlights | Unlimited* | Unlimited* |
| [AI assistant](/stella) messages | — | **Unlimited*** |
| File uploads | **100 MB**/month | **10 GB**/month |
| Single file size | 100 MB | **300 MB** |
| [Web archive](/permanent-copy) | — | **Unlimited** |
| [Reminders](/reminders) | — | **Unlimited** |
| [Annotations](/highlights#annotating) | — | **Unlimited** |
| [Full-text search](/using-search) | — | **Yes** |
| [Broken link checker](/broken-links) | — | **Yes** |
| [Duplicate finder](/duplicates) | — | **Yes** |
| [Automatic backups](/export#automatic-daily-backups) | — | **Yes** |

\* Up to 100 highlights per bookmark  
\*\* Fair usage policy applies

<AccordionGroup>
  <Accordion title="What counts toward the monthly file upload limit?">
    Only files you upload manually (PDFs, images, documents, etc.) count toward the limit. Saving bookmarks, web archive copies, and bookmarks to files hosted elsewhere do not use your upload quota.
  </Accordion>

  <Accordion title="What happens when I reach the upload limit?">
    You won't be able to upload new files until the next billing cycle. Existing files remain accessible.
  </Accordion>
</AccordionGroup>
