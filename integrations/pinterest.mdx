[Pinterest](https://www.pinterest.com/) is a visual discovery platform for saving ideas and inspiration. With this integration you can import your existing pins and set up automatic sync so new pins are saved as bookmarks.

## Importing existing pins

Pinterest doesn't have a built-in export. Use [Pinback](https://pinbackit.github.io/), a free third-party tool that downloads your pins as an importable file.

<Steps>
	<Step>
		Download your pins with [Pinback](https://pinbackit.github.io/)
	</Step>
	<Step>
		Go to [Settings → Import](https://app.raindrop.io/settings/import)
	</Step>
	<Step>
		Upload the file
	</Step>
</Steps>
