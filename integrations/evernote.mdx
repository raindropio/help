[Evernote](https://evernote.com/) is a note-taking app with web clipping. With this integration you can import your existing web clips and set up automatic sync so new notes flow between both apps.

## Importing existing web clips

Only notes with a source URL (created via Web Clipper) are imported — regular text notes are skipped.

<Steps>
	<Step>
		[Export your notes](https://help.evernote.com/hc/en-us/articles/209005557-Export-notes-and-notebooks) in ENEX format. Select only **Created Date**, **Tags**, and **Source URL** — other fields are ignored
	</Step>
	<Step>
		Go to [Settings → Import](https://app.raindrop.io/settings/import)
	</Step>
	<Step>
		Upload the ENEX file
	</Step>
</Steps>

Learn more about [importing bookmarks from other apps](/import).
