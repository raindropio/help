[Notion](https://www.notion.so/) is an all-in-one workspace for notes, docs, and databases. With this integration you can import your existing web links and set up sync so new bookmarks flow between both apps.

## Importing existing items

<Steps>
	<Step>
		[Export your entire Notion workspace](https://www.notion.com/help/export-your-content#export-your-entire-workspace)
	</Step>
	<Step>
		Unzip the archive and find the `My Links.csv` file inside
	</Step>
	<Step>
		Go to [Settings → Import](https://app.raindrop.io/settings/import)
	</Step>
	<Step>
		Upload the file
	</Step>
</Steps>

Learn more about [importing bookmarks from other apps](/import).
