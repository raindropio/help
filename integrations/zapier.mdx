[Zapier](https://zapier.com/apps/raindropio/integrations) is a no-code automation platform. With this integration you can connect your bookmarks with thousands of other apps and build custom workflows.

## Triggers and actions

Each workflow (a **Zap**) starts with a trigger and runs one or more actions. Mix Raindrop with any other app Zapier supports.

| Triggers (when…) | |
|---|---|
| New item | A new bookmark is created |
| New item moved to collection | You move an item to a collection |
| New favorited item | You favorite an item |
| New tagged item | You tag an item |

| Actions (then…) | |
|---|---|
| Create item | Save a new bookmark |
