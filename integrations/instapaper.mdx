[Instapaper](https://www.instapaper.com/) is a read-later app for saving web articles. With this integration you can import your existing items and set up two-way sync so new articles flow between both apps.

## Importing existing items

<Steps>
	<Step>
		[Download CSV file](https://www.instapaper.com/user) from Instapaper (CSV preserves the most data)
	</Step>
	<Step>
		Go to [Settings → Import](https://app.raindrop.io/settings/import)
	</Step>
	<Step>
		Upload the file
	</Step>
</Steps>
