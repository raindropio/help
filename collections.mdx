As your library grows, finding the right bookmark gets harder.
Collections work like folders — group related items together so you always know where things are.

Each collection has its own icon and layout settings, so you can set up each one the way that works best for its content.
Bookmarks you save without picking a collection land in <Icon icon="inbox" size={16} /> **Unsorted** — a built-in inbox you can sort through later.

## Adding bookmarks to a collection

Every bookmark belongs to one collection. You pick the collection when you [save a bookmark](/bookmarks) — or it goes to **Unsorted** by default.

Already saved bookmarks can be easily moved to the right collection:

<Tabs>
	<Tab title="Web & Desktop">
		Drag a bookmark from the list onto a collection in the sidebar.

		<Frame style={{padding: 0}}>![](/assets/collections-move-drag.png)</Frame>

## Grouping and ordering

Once you have more than a handful of collections, you'll want to keep them organized too. Collections support nesting, reordering, and collapsing.

## Managing a collection

<Tabs>
	<Tab title="Web & Desktop">
        <Steps>
			<Step icon="computer-mouse-button-right">
                Right-click a collection to open the context menu.

		        <Frame style={{padding: 0}}><img src="/assets/collection-context-menu.png" /></Frame>
            </Step>

            <Step icon="pen-to-square">
                **Rename** — type a new name and press **Enter**.
            </Step>

			<Step icon="image">
                **[Change icon](#changing-collection-icon)** — pick a custom icon for the collection.
            </Step>

            <Step icon="trash">
		        **Delete** — bookmarks inside are moved to [Trash](/bookmarks#where-deleted-bookmarks-go), not permanently deleted. Deleting a parent also deletes its sub-collections.
            </Step>
        </Steps>
	</Tab>

	<Tab title="iOS">
		Long-press a collection to open the context menu.

		- **Edit collection** — type a new name and tap **Done**.
		- **Delete collection** — bookmarks inside are moved to [Trash](/bookmarks#where-deleted-bookmarks-go).

		<Frame style={{padding: '20px 0 0 0'}}><img src="/assets/ios-collection-menu.png" style={{maxHeight: '333px'}} /></Frame>
	</Tab>

	<Tab title="Android">
		Swipe a collection to the left and tap <Icon icon="pencil" size={16} /> to open the edit screen.

		- **Rename** — type a new name and tap **Save**.
		- **[Change icon](#changing-collection-icon)** — pick a custom icon.
		- **Delete** — bookmarks inside are moved to [Trash](/bookmarks#where-deleted-bookmarks-go).
	</Tab>

## FAQ

<AccordionGroup>
	<Accordion title="What happens to bookmarks when I delete a collection?">
		They are moved to [Trash](/bookmarks#where-deleted-bookmarks-go), not permanently deleted. You can restore them from there.
	</Accordion>

	<Accordion title="Can I save a bookmark into multiple collections?">
		A bookmark can only live in one collection. To have it in two places, open the target collection, click <Icon icon="plus" size={16} />, paste the URL, and save — this creates a separate copy.
	</Accordion>

	<Accordion title="Can I set a default layout for all collections?">
		Yes. Go to [Settings → App](https://app.raindrop.io/settings/app) and change **Default layout**. New collections will use this layout automatically.
	</Accordion>

	<Accordion title="How do I delete all empty collections at once?">
		On Web & Desktop, right-click **My collections** in the sidebar and choose **Delete all empty collections**.
	</Accordion>
</AccordionGroup>
